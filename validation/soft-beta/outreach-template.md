# Soft-beta outreach template

Lower-friction than the formal-interview email. Different framing.

---

**Subject:** Quick favor — a portfolio tool I built

Hey [name],

I shipped a small CLI that analyzes a GitHub repo and tells you how
it reads to a recruiter in 2026 (when AI-generated content has eroded
resume trust). It's free, local-first, takes 10 seconds to run.

Would you give it a spin against one of your repos? Two-step install
and one command:

```bash
npm install -g @tucaken/signal-cli
tucaken-signal /path/to/your/repo
```

If anything surprises you — output that lands as useful, output that
lands as wrong, or anything you'd want different — send me one
paragraph back. No formal commitment, no scheduled time.

I'm running it past ~15 engineers I trust before doing any public
push. Your reaction is one of the things shaping what ships next.

Thanks,
[your name]

P.S. Source at https://github.com/Nelson-Lamounier/tucaken-signal
if you want to look first.

---

## Notes for sending

- Send to 10–15 from your network. Mix of stages and roles.
- DM / Slack is fine; format the install snippet for the channel.
- Don't include "the AI-transparency framing" or other internal
  vocabulary. They're using it cold, you want their first read.
- Compensate for time only if it would be weird not to in your
  relationship (small gift card to a close colleague, nothing
  to a casual acquaintance).
